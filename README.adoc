== 이름 도식

`hosts`::
   `homeConfigurations` 나 `nixosConfigurations` 의 이름에는 hostname을 사용할 것.

=== 모듈 內 디렉토리 이름 도식

`programs`::
    설치할 프로그램 및 그 설정

`packages`::
    위와 유사하나 설정 선언이 없음

`modules`::
    모듈 선언

`services`::
    systemd 서비스

`hardware`::
    ** 하드웨어 관련
    ** bootloader, fstab, initrd, hostname, kernel 등 포함
    ** `hardware-configuration.nix` 에 들어가는 내용.

`configs`::
    모듈의 옵션 설정 (위에 해당하지 않는 값)

`lib`::
    공용 함수

`resources`::
    `nix` 파일이 아니거나, 단순 참조 파일
